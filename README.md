## Description
Adds an in-game Creative Menu overlay that lets players spawn any item
directly into the world as a dropped item at the player's feet, in any
quantity.

Press C while playing to open or close the menu. The game's cursor is
released while the menu is open so you can click items freely.

## Features
- Searchable item list: all items from ItemMap, sorted alphabetically.
  Type in the search box to filter by name.
- Selected item display: clicking an item shows its icon and name in
  the bottom panel.
- Quantity selector: type a number directly into the field, or use the
  − and + buttons to increment or decrement by one. The value is
  clamped to a minimum of 1.
- Spawn button: disabled until an item is selected. When pressed,
  spawns the full quantity as a single dropped-item stack at the
  player's current position.

## Installation
Create a mods folder within lucid blocks directory and place Recipe_Browser_v0.1.pck inside.

## Notes
- To uninstall, delete the mods/creative_menu/ folder (or its .pck).
